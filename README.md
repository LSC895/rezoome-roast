# 🔥 Rezoome Roast

  AI-powered resume roasting and fixing platform for Gen-Z job seekers.

  ## Tech Stack
  - React + TypeScript + Vite
  - Supabase (auth, database, edge functions)
  - Tailwind CSS + shadcn/ui

  ## Getting Started
  1. Clone the repo
  2. Copy `.env.example` to `.env` and fill in your Supabase credentials
  3. Run `bun install`
  4. Run `bun dev`
